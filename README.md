# VIII - Deployment - HARKKA

The goal of this project was to test a JavaScript library. The focus was on writing unit test, identifying possible issues in the code, and measuring code coverage.
The project included setting up automated testing using GitHub Actions and generating coverage reports using Coveralls.

## Coverage

The project achieved approximately 61% coverage, meeting the target of at least 60%.

Coverage is tracked using Coveralls.

## GitHub Actions

GitHub Actions is used to automatically run tests and coverage on every push.

## Tools used in this project:

• Node.js version 22.20.0  
• Mocha – version 11.7.5  
• Chai – version 6.2.2  
• C8 – version 11.0.0  
• Coveralls – version 3.1.1  
• GitHub Actions

## Installation

1. Clone the repository
2. Go to the project directory
3. Install dependencies  
   npm install
4. Run tests  
   npm test
5. Run coverage
   npm run coverage

## Issues

The issues founded are documented in the GitHub Issues

## Conclusion

The project achieved the required coverage and identified several issues in the library.

While many functions were tested successfully, further testing would improve reliability.
